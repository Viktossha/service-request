# Service Request

Небольшой Laravel-модуль для демонстрации услуг ателье и приёма заявок.

В проекте реализованы:

- адаптивный блок с тремя услугами;
- форма заявки с серверной валидацией;
- сохранение заявок в SQLite;
- сообщение об успешной отправке;
- адаптивная страница со списком заявок.

## Запуск проекта

### Требования

- PHP 8.3 или новее;
- Composer;
- Node.js и npm;
- расширение PHP для SQLite.

### Установка

```bash
git clone https://github.com/Viktossha/service-request.git
cd service-request
composer setup
```

Команда `composer setup` установит зависимости, создаст `.env`, сгенерирует ключ приложения, выполнит миграции и соберёт frontend.

### Запуск

```bash
composer dev
```

После запуска приложение будет доступно по адресу:

```text
http://127.0.0.1:8000
```

Страница со списком заявок:

```text
http://127.0.0.1:8000/applications
```
